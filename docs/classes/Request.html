cancel

  • cancel(reason?: string | number): void
    • Defined in src/request.ts:100

    The queues can call this if the request has been on the queue for too long. Original requestors can call this as well from the CancellableEventEmitter. Queues will give us the timeout they applied. Through the emitter we may get a reason. If we do, the request will be reported as 'Cancelled' to the original requestor with that reason(promise or callback). If we don't get a reason, the request will be cancelled silently.

    • Optional reason: string | number
